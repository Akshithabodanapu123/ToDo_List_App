# To-Do List Application

## Project Description

The To-Do List Application is a Python-based command-line program that helps users manage their daily tasks efficiently. Users can add tasks, view the task list, delete completed tasks, and exit the application.

## Features

* Add new tasks
* View all tasks
* Delete tasks
* User-friendly menu interface
* Simple and efficient task management

## Technologies Used

* Python

## How It Works

1. The user selects an option from the menu.
2. Tasks can be added to the list.
3. All tasks can be viewed at any time.
4. Tasks can be deleted when completed.
5. The user can exit the application safely.

## Sample Operations

* Add Task
* View Tasks
* Delete Task
* Exit Application

## How to Run

1. Save the file as `todo_list.py`
2. Open a terminal or command prompt.
3. Run the following command:

python todo_list.py
